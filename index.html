Champlain Computer Science Club!

Who?

Champlain Students gather from all over the world (campus) to meet for computer science club!

Club Leadership:

  • Mason Lee
  • Ryan Buck

What do we do?

Anything to do with programming!

Here are some of the activities we have done so far!

  • Scratch Night A beginner-friendly night of chaotic creativity using MIT Scratch. Make weird little games, animations, or cursed cats.
  • PvP Coding Head-to-head coding battles. Timed challenges, obscure problems, and bragging rights on the line.
  • Roblox Night Build, script, or just explore Roblox games together. Lua-fueled chaos encouraged.
  • Code Golf Write the shortest, most cursed code possible to solve a problem. Bonus points for unreadability.
  • Esolang Night Try programming in languages that were never meant to be practical. Brainfuck, Befunge, and other monstrosities welcome.
  • Processing Night Use the Processing language to make cool visuals, generative art, or glitchy experiments. No art degree required.
  • Code Swap Write some code, then swap with someone else and try to make sense of their mess. Beautiful chaos ensues.
